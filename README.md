# Pet Adoption Platform
Features
User Authentication (Adopter, Shelter)
    Register & Login (Adopter / Shelter)
    Role-based access

Pet Listings
    View all pets (Adopter)
    Add/Edit pets (Shelter)

Apply for Adoption
    Apply for pet adoption (Adopter)
    Approve / Reject applications (Shelter)

Favorites
    Adopters can add/remove favorite pets

Reviews
    Adopters can give reviews & ratings


Tech Stack
Frontend: React + Tailwind
Backend: Node.js + Express
Database: MongoDB
